家喵二三事

2022年４月８日 开始发售！

介绍

带着某个使命
从天界来到人间界的薄荷
在途中体力不支倒下了

一介平凡大学生的主人公
偶然发现并帮助了她
也因此得到了她的青睐
虽然半推半就地被她住进了家里
但也被有点自大却又可爱率真的薄荷所吸引

正当被像动物一样有着耳朵和尾巴的的天使称作「主人」
享受与像宠物一样撒娇的薄荷一起的要化掉般甜蜜又热闹
卿卿我我、打情骂俏的生活之时

兽人恶魔『莉莉』与薄荷的妹妹天使『可可』
也加入进了这样的生活当中
闹剧与欢乐的骚乱要来咯――！

NAME

薄荷

CAST
飴川紫乃

身为半兽天使的女孩子。
有着动物耳朵和尾巴的天使，虽然长有翅膀，但是却没有“天使之环”，很介意自己翅膀小的事情，平常都是背着假的玩具翅膀。不擅长使用天使们都会的“魔法”，还没法做到独当一面。

为了寻找“某个东西”而从天界下凡，苦苦寻而不得，中途倒在了路上被主人公所救起。之后住进了主人公家，把主人公亲切地叫做“主人”，像宠物一般亲近。

性格有点小任性，喜欢恶作剧，为了报答主人公的恩情而努力地想照顾主人公，却总是反过来被照顾了。

NAME

莉莉

CAST
立花沙羅

身为半兽恶魔的女孩子。
有着动物耳朵和尾巴的恶魔，背上长着一对黑色的翅膀。能使用恶魔的能力来“消除存在感”。身材很好，经常拿这点来逗薄荷。

怀揣着自己的伙食自己挣的想法，便隐藏起自己的真实身份去打工挣钱。喜欢开心的事情，基本上整天都是乐呵呵的样子，但也会突然展露出认真的模样与人交谈。

NAME

可可

CAST
水野七海

身为半兽天使的女孩子。
有着动物耳朵和尾巴的天使，与薄荷不一样，还有着“天使之环”。作为天使十分优秀，能自如地使用魔法。但是和薄荷一样，仍然还不能独当一面。

姐姐薄荷临行前说去一趟地上，却迟迟没有回来，因为担心所以自己也来到了地上。知道姐姐的目的，正奋力想办法改变姐姐的想法。
